# HTML → PDF Converter – Benutzerhilfe

## Installation
1. Installieren Sie wkhtmltopdf:
   https://wkhtmltopdf.org/downloads.html

2. Stellen Sie sicher, dass wkhtmltopdf im PATH liegt.

## GUI-Nutzung
- Starten Sie das Programm ohne Parameter.
- Wählen Sie HTML-Dateien aus.
- Klicken Sie auf „Konvertieren“.

## CLI-Nutzung
``` bash
python main.py --input <pfad> [--output <ordner>] [--silent] [--log <datei>]
```
